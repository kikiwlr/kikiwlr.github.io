ES6

2020-09-22

1.预解析：

JS代码在执行之前，会对代码进行预解析，寻找作用域中的var 和function ，然后对其进行事先声明，在从上到下执行代码。这就是一个预解析的过程。

var 未定义变量而使用是undefined.

let未声明的话而使用的话会报错。

let声明的变量不存在预解析

2.定义赋值

var 两次定义，后会覆盖前面的。

let 的话，报错，标识符被声明

let声明的变量不允许重复（在同一个作用域内）

3.ES6引入了块级作用域

使用{}、if（）{}形成块级作用域，for循环括号内的声明的变量，循环体的使用；

在块级作用域内部，变量只能先声明再使用；

在块级作用域内，

var定义的变量，在作用域外可以访问到；

let定义的变量，在作用域外不能访问到，会报错；

4.const

用来声明常量,声明的常量不允许重新赋值，所以声明时要初始化。

const n = 1;

5.解构赋值：

变量的

var a = 1;

var b = 2;

var a = 1,b = 2;

数组的

var [a,b,c] = [1,2,3];

let [a=111,b,c] = [,123,];

对象的

let {foo,bar} = { foo : ‘ hello ‘ , bar : ‘ hi ‘};

与赋值顺序无关，完全按照变量的名称顺序来赋值；

对象属性别名：

别名存在时，原来的名字无效；

let{foo : abc,bar} = {bar: ‘hi’ , foo:’nihao’};

console.log(foo,bar);

let {cos , sin , random} = math;

一次性将内置对象赋值；

字符串的解构赋值；

变量的

let [a,b,c,d,e] = “hello”;

将字符一个一个的拆开；

console.log(“hello”.length);

对象的：

let {length} = “hi”

console.log(length);

2对象的解构赋值指定默认值：

let {foo:abc = ‘helllo’, bar} = {bar: ‘hi’};

6

includes():判断字符串中是否包含指定子串；true\false

‘hello’.includes(‘llo’,2)；

参数（子串，从第几个开始匹配）——url地址

startsWith()、endsWith():判断字符串是否以特定的子串开始、结束。

let url = ‘admin/index.php’;

console.log(url.startsWith（’adi’）);

7.模板字符串：

将字符串和变量拼接起来以前通过运算符“+”来实现，过多还要用“\”来表示换行；

ES6中，可以将反引号（``）将内容括起来，在反引号中，可以使用${}来写入需要引用到的变量；

8.调用原型对象

在ES6中，通过使用super可以调用某个对象的prototype对象的方法或获取参数

9.迭代

迭代器iterator

for…in 语句以原始插入顺序迭代对象的可枚举属性，for…in是用来循环遍历属性；

遍历出的是自身和原型上的可枚举非symbol属性，但是遍历不一定按照顺序（for…in在ES5中就已经出现）；

for…of 语句遍历可迭代对象定义要迭代的数据，也就是说，for…of只可以循环可迭代对象的可迭代属性，不可迭代属性在循环中被忽略了。（for…of是ES6才提出来）

10.class

ES6中引入了 Class（类）这个概念，作为对象的模板。通过class关键字，可以定义类。
